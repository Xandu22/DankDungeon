Proposal

We will be pursuing the development of an online school management resource that utilizes a public government database for K-12 students to showcase several types of edge cases.

Plan to develop an EER diagram showcasing the student categorization including disjoint and multiple inheritance examples.

Convert the EER diagram into a relational diagram.

Expanding on this, we plan to implement the relational diagram into our database via SQL statements and produce views through triggers.

Ideally a stored procedure will be made to add user data into appropriate tables and even create new ones if a trend starts to appear.

Finally we will produce a simple front end web site for users to see different views of existing data or insert their own.
